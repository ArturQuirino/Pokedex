# Pokedex
Pokedex para o Minas Coders


## Passos para criação

### Frontend
`npx create-react-app pokedex`
`npm install react-router-dom`
`npm install axios`
`npm install --save-dev jest`

### Backend
`npm init`
`npm install express`
`npm i -D nodemon`
`npm install axios`
`npm install cors`

`npm install mongodb`
`npm install dotenv`

API's de referência:
https://pokeapi.co/docs/v2#pokemon
